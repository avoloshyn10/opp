Open Panzer Pedia REST Client Guide

The OPP project comes with an embedded REST client which can be used to manage units and their available RDF resources.

Initial Screen

Navigating to http://localhost:8080/ will open the default screen which lists all units available in game database. The list shows Unit ID, name, country and military class that the unit belongs. Queried show if unit has been processed or not.

Unit Information Screen

Clicking on a unit name on the main screen will show details about RDF data and Queries used for this unit

Unit Details Screen

Clicking on Unit Details button will show all parsed RDF details available for this unit.

Unit RDF Data Screen

Clicking on Unit RDF Data button will show the entire RDF resource stored on local database. This will include all languages available and all subject/predicate/object triples

Unit Edit Screen

Clicking on Edit unit resource will show the unit edit screen where it's possible to either select an already found resource, or add a custom resource for this unit. Force Refresh will instruct the server to redo the search queries and reload the RDF data.

If a unit has status No for Queried it will be created based on resource entered by user.
